Background Workers & External API connections‚Äã

For this section you will need a Twilio SID & Token and/or a Slack Token, depending on if you want to send Texts, DMs, or both.

Twilio‚Äã

To create a new Twilio account, the SID & Token are in your console on the right hand side.

Once you have an account, you‚Äôll need a number that can send texts, https://www.twilio.com/console/phone-numbers/incoming.

Slack‚Äã

To create a Slack token, go to https://slack.com/api, create a new application, add a bot from the ‚Äúbasic information‚Äù add features & functionality section, and install the app in your workspace. After doing so, you‚Äôll get two tokens back, one of which is the format "xoxb-*".
