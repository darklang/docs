Getting started

Note that you can also contribute without coding!

This guide will get you making your first PR to the Dark repo. It helps you set up the repo, write your first test, and make your first pull request. After that, it will guide you on how to work on bigger things, especially the current big project to port the backend to F#.

If you'd like help contributing, you can book a pairing session with Paul Biggar to help.

Before you do that, it's helpful to join the Darklang Slack. All discussions are in the #general room.
